活動介紹

有滿腦子的 APP 創意需要實現嗎？Firefox OS APP 競賽提供全新 APP 藍海戰場，是你大展身手的最佳機會！Firefox OS自推出以來，於開發者之間蔚為一股新浪潮，而 Firefox Marketplace 更是有心擴大 APP 創業者的版圖。Firefox OS 完全以 HTML5 技術打造，以 JavaScript、CSS 等技術搭配全新 WebAPI，讓 HTML5 的 應用程式擁有更多可能性。Firefox OS 為行動 APP 開發者開拓一個全新的戰場，而擅長 HTML5 的網站開發者也找到快速切入跨平台 APP 市場的方式。

竭誠歡迎熱愛開放 Web 的 Firefox校園大使們，一同落實 APP 夢，在學生時代，找到一片寬廣的藍海。請大家踴躍參加！ 2013 年一場您絕不能錯過的校園大使 APP 創意競賽，現正隆重登場！
