活動辦法

各區校園大使至少需開發 1 個APP，無開發數量上限。請於 10/21 – 10/31 繳交企劃書，並請於 12/01前完成作品並繳件附上實作連結。12/21 成果發表日進行現場發表。最終由成果發表日之評審分數與網路票選票數決定優勝隊伍。參賽 APP 作品類型不限，舉凡遊戲、實用、生活、社群、數位內容…等，均可參賽。

  • 企劃書繳交

    企劃書檔案格式：Google document
    請各區協調一位代表於 10/31 以前至官方網站 企劃書繳交 專區，套用「進撃のWeb企畫書範本」 填寫企劃內容並提供企劃書連結。(企劃書內容不得超過10頁)。

  • 作品繳交

    請參賽者於 11/21~12/01 完成作品，前往成果發佈專區上傳，並提供作品之介紹影片或投影片連結。
    作品繳件形式：GitHub 連結或 APP Host 之 Server 連結。

  • 網路票選

    投票者藉由成果展示專區了解各區大使作品並進行網路人氣投票。
    網路票選期間為 12/05 – 12/20

  • 成果展示 LIVE Show

    請參賽者於 12/21 出席成果展示活動，進行作品之成果展示。
